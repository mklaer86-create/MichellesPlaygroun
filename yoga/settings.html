GitHub sync

Saving in the Pose Library or Sequence Builder commits your changes straight to this repository, so the app updates for everyone. That requires a GitHub personal access token, stored only on this device.

Use a fine-grained token scoped to only this repository (mklaer86-create/MichellesPlaygroun) with Contents: Read and write permission — nothing else. Treat it like a password: never paste it into a shared or public computer, and set an expiration date when you create it.
Create one at GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
